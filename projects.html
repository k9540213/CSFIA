My Course Projects

CS 3350: Employee Management System

This was my final project for Software Engineering. I led a small team to design and build a desktop application for managing employee records. The system was built using Java, with a graphical user interface created with JavaFX. Key features included the ability to add, edit, and delete employee records, search and filter the employee list, and manage departmental information. The project emphasized object-oriented design principles, version control with Git, and following an agile development methodology.

Personal Portfolio Website

The website you are currently viewing. I designed and built this multi-page portfolio from scratch using HTML and Tailwind CSS. The project involved creating wireframes and a sitemap, focusing on responsive design, and ensuring a clean, professional user interface to showcase my academic work.

Information Literacy Analysis

This project involved a deep dive into effective research strategies. I selected a topic and used three individual search engines and three meta-search engines to gather information. The core of the project was applying Boolean operators (`AND`, `OR`, `NOT`, `" "`) to create precise queries and evaluating source credibility using the CRAAP test.
